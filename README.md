# MWI Enhancing Prosperity

Calculate profits and ROI for item enhancing in Milky Way Idle. Live prices from the official MWI API, volume averaged from market observatory.

## Usage

Open the [live site](https://star-in-a-galaxy.github.io/mwi-enhancing-prosperity/) or serve locally:

```bash
python -m http.server 8000
```

Note: Most browser will support without python host.

Configure your gear, select pricing modes, and browse the results table. Click any row for a full breakdown.
